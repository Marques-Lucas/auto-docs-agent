# 2026-07-02 — Setup do sistema de organização do projeto

## O que foi decidido
Definido o sistema de organização/memória deste projeto, controlado pelo
`CLAUDE.md` na raiz. A partir de agora, decisões relevantes viram arquivos de
sessão e pendências ficam num arquivo único.

## Detalhes técnicos
- `CLAUDE.md` (raiz) — fonte da verdade das regras de organização.
- `/sessions/` — um arquivo por sessão com decisão relevante, nomeado
  `AAAA-MM-DD-titulo-curto.md`.
- `/pendencias.md` (raiz) — arquivo único de pendências em formato checklist
  (`[ ]` aberto, `[x]` concluído; nunca apagar linhas, só marcar).
